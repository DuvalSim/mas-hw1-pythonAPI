# mas-hw1-pythonAPI

## Repository presentation
This is a test project to build a REST API interracting with firebase real time database.
The file test.py contains basics instructions to check that the library is working and can retrieve data from our database.

## How to run
* Clone this repository
* Setup a virtual environment
* Install the only required dependency : [pyrebase](https://github.com/thisbejim/Pyrebase)
`pip install pyrebase4`
* Complete the config object in test.py with the informations from your firebase application.
* Run with `python test.py`
